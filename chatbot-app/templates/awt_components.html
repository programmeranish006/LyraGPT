What is AWT?

The Abstract Window Toolkit (AWT) is Java's original platform-independent windowing, graphics, and user-interface widget toolkit. It provides a set of components for building graphical user interfaces (GUIs) in Java.

Note: While Swing and JavaFX are more modern alternatives, AWT is still widely used and provides a lightweight foundation for GUI development.

Container Components

Containers hold other components and manage their layout.

The top-level window for your application. Every AWT application must have at least one Frame.

Input Components

Components that allow users to input text and data.

Single-line text input field. Perfect for names, emails, passwords, etc.

‚Ä¢ setText(String text)
‚Ä¢ getText()

‚Ä¢ addActionListener(ActionListener l)

Selection Components

Components that allow users to select from predefined options.

Dropdown menu showing one selected item at a time. Compact for long lists.

Key Methods:
‚Ä¢ addItem(String item)

Shows multiple items with optional multi-selection support. Takes up more space but shows all options.

‚Ä¢ add(String item)

‚Ä¢ addItemListener(ItemListener l)

Graphics & Layout

Understanding how to draw and organize components.

Common Layouts:
‚Ä¢ FlowLayout - Left to right, top to bottom
‚Ä¢ BorderLayout - North, South, East, West, Center
‚Ä¢ GridLayout - Grid of equal-sized cells
‚Ä¢ GridBagLayout - Complex layouts with constraints

Graphics Class

Used within Canvas.paint() to draw shapes, text, and images.

Common Methods:
‚Ä¢ drawLine(int x1, int y1, int x2, int y2)
‚Ä¢ fillRect(int x, int y, int w, int h)
‚Ä¢ fillOval(int x, int y, int w, int h)
‚Ä¢ drawString(String text, int x, int y)

Event Listeners

Handle user interactions like clicks, key presses, and window events.

Common Listeners:
‚Ä¢ ActionListener - Button clicks
‚Ä¢ ItemListener - Checkbox/Choice selection
‚Ä¢ WindowListener - Window events
‚Ä¢ MouseListener - Mouse events

Quick Start Guide

Basic AWT Application Structure:

import java.awt.*; import java.awt.event.*; public class MyApp extends Frame implements ActionListener { public MyApp() { setTitle("My Application"); setSize(400, 300); setLocationRelativeTo(null); setLayout(new FlowLayout()); Button btn = new Button("Click Me"); btn.addActionListener(this); add(btn); addWindowListener(new WindowAdapter() { public void windowClosing(WindowEvent e) { System.exit(0); } }); setVisible(true); } public void actionPerformed(ActionEvent e) { System.out.println("Button clicked!"); } public static void main(String[] args) { new MyApp(); } }
